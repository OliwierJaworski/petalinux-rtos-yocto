## About

This repository is a collection of projects specifically based on embedded FPGA systems,
such as the PYNQ-Z2 and Avnet ZUBoard 1CG.

---

## Features

- PetaLinux-based embedded Linux support
- RTOS and bare-metal compatibility
- Yocto build system integration
- Standalone firmware and hardware projects

---

## Installation

Clone the repository:

```bash
git clone https://github.com/OliwierJaworski/petalinux-rtos-yocto.git
```

Enter the project directory:

```bash
cd petalinux-rtos-yocto
```

Read, build or run according to the target platform.
